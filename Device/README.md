# Device Reports

## DeviceReport_ImportCSV.ps1
Import a CSV of device hostnames and get back a report with last user, SCCM client status, online status, last logon date, IP/MAC address, serial number, OS version, and OU.

## DeviceReport_SinglePC.ps1
Quick single-device lookup that returns the same device info interactively, plus BitLocker recovery key and admin credentials.
